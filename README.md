# TaskNest 🗂️

TaskNest is a lightweight, user-friendly task management system built using PHP, MySQL (via XAMPP), Bootstrap, and CSS. Designed for simplicity and responsiveness, TaskNest helps users organize, track, and manage tasks efficiently through a web interface.

---

## 🚀 Features

- ✅ Add, edit, and delete tasks
- 📌 Mark tasks as complete or pending
- 🗓️ Due dates and priority levels
- 🧾 Categorize tasks
- 📱 Responsive UI with Bootstrap
- 💾 Data stored using MySQL (XAMPP)

---

## 🛠️ Tech Stack

- PHP
- MySQL (XAMPP)
- HTML5 & CSS3
- Bootstrap 5
- JavaScript (optional for interactivity)

---

## 📦 Installation

1. **Clone the repository:**

   "git clone https://github.com/yourusername/TaskNest.git"
Move the folder to your XAMPP htdocs directory:

mv TaskNest /xampp/htdocs/
Start XAMPP and run Apache and MySQL.

Import the SQL database:

Go to http://localhost/phpmyadmin

Create a new database (e.g., tasknest_db)

Import the provided .sql file (if included in the repo)

Access the app in your browser:
http://localhost/TaskNest
